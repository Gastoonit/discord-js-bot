### Discord.js Handler Bot
***
Simple structure for your message commands in discord.js
## Install:
```
$ npm install
```
## JSON:
```json
{
    "BOT_TOKEN": "DATA_BOT_TOKEN",
    "DEFAULT_PREFIX": "!"
}
```
## Clone:
```git
git clone https://github.com/Gastoonit/discord.js-bot.git
mv discord.js-bot/* .
```
